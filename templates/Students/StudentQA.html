Student Q&A

  • question_answerHow Can I Set My Avatar?
    You can set your avatar by going to Tools menu, then to "Avatar Selection", and from there you can pick a new avatar.
  • question_answerHow Can I Remove the Class Leaderboard from My Course Home Page?
    If you wish to disable the class Leaderboard you can go to Tools menu and select Preferences. Here you will find toggles to disable viewing the class Leaderboard.
  • question_answerHow Can I Enable/Disable the Display of Class Average in My Achievement Page?
    If you wish to disable the class Achievements you can go to Tools menu and select Preferences. Here you will find toggles to disable viewing the class Achievements.
  • question_answerHow Can I Enable/Disable the Display of Class Skills?
    If you wish to disable the Class Skills you can go to Tools menu and select Preferences. Here you will find toggles to disable viewing the class Skills.
  • question_answerWhere Can I See What Badges Are Available in This Course?
    To see Course Badges go to Course Information menu and select Course Badges. Here you can view all the available Badges, and how to earn them.
  • question_answerHow I Can Earn Virtual Currency?
    To see the rules on Earning Virtual Currency go to the Course Information menu and select Currency Rules.
  • question_answerWhere To Check How I Have Earned my Virtual Currency?
    To check how much virtual currency you have, go to your Achievements menu.

    To check your earned virtual currency transactions, go to the Tools menu and select Your Earning Transactions.
  • question_answerWhat Is the Status of My Purchases from the Course Shop?
    To check the status of your Earned Transactions go to the Tools menu and select Your Spending Transactions.
  • question_answerWhat Is the Difference Between Warm-Up Challenges and Serious Challenges?
    The difference between a Warm-Up and a Serious Challenge is that a Warm-Up Challenge is intended to be used as a practicing and self-check tool. Warm-Up Challenges may be re-taken as often as needed, and often give credits from successful attempts.

    It is highly recommended to use Warm-Up Challenges to study for Serious Challenges or Exams in your course. A Serious Challenge is intended to grade your knowledge and skills on a topic for a grade.
  • question_answerWhat Are Parson's Problems and How Do I Solve Them?
    Parsons problems are problems in which you have to order code "code lines" in a correct order. You have to drag and drop the given "code blocks" onto the silver-grey box on the right. Please pay attention to indentation, as it may be graded.

    When you press the "Get Feedback" button, each wrong line will be highlighted in red to indicate that the line is wrong. If your order is considered correct the code will be highlighted in green. Note that each time you press the "Get Feedback" button, if the code is incorrect this will halve the points.
  • question_answerHow Is a Parson's Problem Graded?
    Each time you press the "Get Feedback" button, if the code is incorrect this will halve the points. If the problem was correct, then you will receive full credit.

    If “Code Indentation” is set to “true” by the instructor, indentation errors in the your solutions are counted against you at the rate of 1 point deduction per N lines of code divided by half, i.e. (1/N)*(1/2).

    Each "code line" or "code block" error is a deduction of (1/N). Each "code line" or "code block" under or over the required number of correct lines is also a deduction of (1/N).
  • question_answerWhat Is the Meaning of the Different Colors in My Progress Bar in the Achievement Page?
    The progress bar in the Achievements page consists of four parts displayed in different colors:
    • The course points earned so far (purple).
    • The amount of course points that can be earned in the future (yellow).
    • The amount of points already lost (red).
    • The learning predictor shows the total amount of points that would be earned in this course if you maintain your current performance (dark blue).
