  • question_answerWhat Is the Difference Between Warm-up Challenges and Serious Challenges?
    The difference between a Warm-Up Challenge is that a Warm-Up Challenge is intended to be used as a self-check tool. Warm-Up Challenges may be retaken as often as needed, and often give credits from successful attempts.

    It is highly recommended to use Warm-Up Challenges to study for Serious challenges or Exams in your course. A Serious Challenge is intended to grade your knowledge and skills on a topic for a grade. You cannot retake a Serious Challenge.
  • question_answerWhat Are Parson's Problems and How Do I Solve Them?
    Parsons problems are problems in which you have to order code in such way that if the code were ran on a computer it will be sucessful. Drag and drop the code blocks onto the silver-grey box on the right. Please pay attention to indentation, as it may be graded. When you click on the feedback button, the maximum amount of points for the question will be halved.

    Each press of the feedback button after the initial press,will also halve the points. Each wrong line in your solution when you press “Get Feedback” will be highlighted to indicate that the line is wrong. Indentation errors will also be highlighted. A Parson problem is considered correct when the system highlights all the code in green.
  • question_answerHow Is a Parson's Problem Graded?
    If you click the “Get Feedback” link and you did not get the problem correct, the max number of points for the problem is halved. If the problem was correct then the student will receive full credit.

    The maximum points are halved each time you click the “Get Feedback” link. If “Code Indentation” selector is set to “true” by the instructor, indentation errors in the your solutions are counted against you at the rate of 1 deduction per N lines of code divided by half. In mathematical notation (1/N)*(1/2).

    Each code line or code block error is a deduction of 1/N Each code line or code block over the required number of correct lines is a deduction of (1/N) Each code line or code block under the required number of correct lines is a deduction of (1/N)
  • question_answerWhat Is the Meaning of the Different Colors in My Progress Bar in the Achievement Page?
    The progress bar in the student dashboard consists of four parts displayed in different colors:
    • The course points earned so far (purple).
    • The amount of course points that can be earned in the future (yellow).
    • The amount of points already lost (red).
    • The learning predictor shows the total amount of points that would be earned in this course if you maintain your current performance (dark blue).
